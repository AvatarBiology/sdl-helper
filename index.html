符合 108 課綱素養評量設計。採用 Google 最新 Gemini 3 多模態模型 驅動，能夠雙軌分析文字與圖表，並提供具備鷹架理論的引導式回饋，讓評量精準而有溫度。

立即開始評量

隱私保護設計・僅需 Google 帳號

AI 不只是閱讀，更是「看見」

深入分析實作歷程與圖表邏輯，解構並分析學生學習的真實樣貌。

多模態的視覺分析

系統不只能讀取完整文字內容，更具備視覺認知能力。能檢查數據圖表的邏輯、實驗照片合理性，並評估排版美學。

智慧歸類與評量

自動識別自主學習屬於「自然科學」、「社會人文」或其他「實作專案」類別，動態調整符合 108課綱素養評量的規準，確保評量信度。

結構化回饋報告

生成五大向度分析雷達圖、清晰列出具體亮點與改進方向，並生成學習歷程檔案上傳時需要的「大學申請百字簡述」。最後附上總結評語和學術倫理檢核。

Core Architecture
Exclusive Tech

突破 Token 限制的
智慧採樣演算法

設計 PDF 結構掃描引擎「視覺權重」過濾器，解決 AI 模型解析圖片之 Context Window 有限的問題。這項技術能完美平衡數據容量上限與分析完整度，確保即使上傳長篇報告，也能精準讀取圖片重點，達成文字與圖片的雙軌分析。

全文語意提取

無損提取全篇 PDF 內的純文字內容，建立基礎語意理解架構，不遺漏任何文字細節、完整分析。

視覺權重運算

針對每一頁計算 Visual_Score。自動偵測圖表密集度，區分「無效頁面」與「精華頁面」。

黃金 15 頁圖表鎖定

強制鎖定封面與目錄，並動態挑選分數最高的關鍵頁面傳送至 Gemini，實現關鍵圖表頁面的精準提取與分析。

簡單四步驟，獲取專業報告

Process Flow

1. 上傳檔案

上傳 PDF 格式的自主學習檔案，建議包含豐富圖表。

2. 等待分析

系統提取文字並智慧挑選視覺精華頁面傳送至 Gemini。

3. 檢視報告

查看雷達圖、具體建議與百字簡述範例。

4. 下載存檔

一鍵下載 PDF 報告存檔，或重新測試下一份檔案。
